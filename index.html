2D LBM D2Q9 – Channel Flow over a Joukowsky Airfoil

Prototype Web CFD solver – JavaScript + HTML canvas (démo visuelle, pas encore le vrai modèle LBM)

Simulation Parameters

Flow Speed (Mach) 0.20
Relaxation Time τ (viscosity) 0.80
Smagorinsky Constant Cs 0.16
Angle of Attack (deg) 5°
Airfoil Thickness (t/c) 0.12
Camber 0.08

Pour l’instant, l’animation représente un champ scalaire synthétique (onde sinusoïdale) pour tester l’affichage. On remplacera plus tard par le vrai champ LBM (ρ, u, v, vorticité, etc.).
