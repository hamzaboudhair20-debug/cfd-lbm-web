2D GPU CFD Prototype – Joukowsky Airfoil + WebGL Field

WebGL2 ping-pong shader field (GPU) + parametric Joukowsky airfoil overlay. Sliders control geometry, shader dynamics and simple aerodynamic estimates.

Simulation Parameters

Flow Speed (Mach) 0.20
Relaxation Time τ (numerical viscosity) 0.80
Smagorinsky Constant Cs 0.16
Angle of Attack (deg) 5°
Airfoil Thickness (t/c) 0.12
Camber 0.08

Aerodynamic Estimates (Thin-Airfoil Theory)

Angle of attack α 5° (0.087 rad)
Lift coefficient CL ≈ 2π α 0.55

CL is estimated using the classical thin-airfoil theory (incompressible, small angles, attached flow). This is an analytical first-order estimate, not yet computed from the GPU field.

The coloured field is a stylised advected/diffused scalar computed on the GPU (RGBA8 textures + ping-pong). It serves as a visual placeholder for future LBM/LES velocity/vorticity fields.
